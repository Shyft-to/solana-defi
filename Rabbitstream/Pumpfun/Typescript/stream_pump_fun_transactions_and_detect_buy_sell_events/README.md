# Stream PumpFun Transactions and Detect Buy/Sell Events with Rabbitstream

This project enables real-time streaming and analysis of PumpFun transactions on the Solana blockchain. By monitoring buy and sell events as they occur, it delivers actionable insights into market activity, helping users track trends, understand trading behavior, and gain a deeper understanding of the PumpFun ecosystem.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Shyft-to/solana-defi.git
   
   cd RabbitStream/PumpFun/Typescript/stream_pump_fun_transactions_and_detect_buy_sell_events
   ```

2. **Install Dependencies:**

    ```bash
    # For example, if using npm
    npm i
    ```

3. **Run the script:**

    ```bash
    # To run the script
    npm run start
    ```
*Note: Please rename the `.env.sample` file to `.env` and input your env details before running the script.*
